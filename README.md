# Walborn Landing Page

Premium modern landing page for Walborn, a Congolese technology company building scalable digital infrastructure and SaaS platforms for Africa.

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Inter Font

## Project Structure
- `index.html`: Main entry point
- `css/`: Custom styles and animations
- `js/`: Interactivity and logic
- `assets/`: Images, logos, and icons

## Deployment to GitHub Pages
1. Push this repository to GitHub.
2. Go to Repository **Settings** > **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Choose the `main` branch and `/ (root)` folder.
5. Click **Save**.
6. Your site will be live at `https://<username>.github.io/<repository-name>/`.
